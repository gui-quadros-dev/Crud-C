# Sistema de Cadastro de Produtos em C

Este projeto foi desenvolvido como prática da linguagem C, com foco em organização de dados, uso de estruturas (struct) e interação com o usuário via terminal. A proposta é simular um sistema simples de gerenciamento de produtos.

## Funcionalidades

O sistema permite:

* Cadastro de produtos
* Leitura de dados inseridos pelo usuário
* Armazenamento das seguintes informações:

  * ID do produto
  * Nome
  * Quantidade em estoque
  * Preço

## Tecnologias utilizadas

* Linguagem C
* Estruturas de dados com `struct`
* Entrada e saída de dados pelo terminal

## Objetivo do projeto

O objetivo principal é reforçar os fundamentos da programação em C, trabalhando lógica, organização de código e modelagem de dados em um contexto próximo de um sistema real.
